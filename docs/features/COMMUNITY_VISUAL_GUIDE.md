## Interaction Patterns

### Click Behaviors
- **Tab**: Switch content
- **Discussion**: Expand/collapse comments
- **Photo**: Open full-screen modal
- **Like**: Toggle like state
- **Pin**: Toggle pin state
- **Delete**: Confirm then delete

### Hover Behaviors
- **Photo**: Show overlay with actions
- **Button**: Translate and shadow effect
- **Discussion**: Subtle highlight

### Loading States
- **Uploading**: Progress indicator
- **Loading photos**: Spinner in grid
- **Posting**: Button shows "Posting..."
